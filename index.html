FAQ

Common questions

Does VoxFlow require an internet connection?

No. VoxFlow runs entirely offline using a local AI model downloaded once at first launch. After that it works with no internet at all.

How is this different from Windows built-in speech recognition?

Windows Speech Recognition requires training and doesn't work well out of the box. VoxFlow uses Whisper AI which is significantly more accurate and works in 99 languages without any training.

Will it work in any app?

Yes. VoxFlow uses a global hotkey and injects text directly — it works in any Windows application including browsers, Word, Slack, VS Code and games.

What are the system requirements?

Windows 10 or 11, 64-bit. A microphone. At least 4GB of RAM recommended for best performance.

Do I get free updates?

Yes. Pay once and get every future update for free. VoxFlow notifies you automatically when updates are available.
